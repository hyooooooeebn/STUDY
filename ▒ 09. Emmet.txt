■■■ Emmet(에밋) Zen coding ■■■

○ 개요
  - 기존에는 zen coding이라는 이름으로 사용되엇다.
  - 이후 구글에서 이 도구를 수용하면서 emmet이라는 이름으로 변경되었다.
  - HTML 생산성 도구(HTML 코딩을 빠르게 하기 위한 플러그인)
  - CSS 선택자 사용(CSS3 지원)
   → HTML을 CSS 선택자를 통해 검색하는 방법 대신
      CSS를 통해 HTML을 생성하도록 하는 개념.

○ 환경 설정
  - 대부분의 툴에서 지원하고 있음
    (※ 대부분의 편집기 내 액션 키 : Ctrl + E)

  - http://emmet.io
   (1) 메인 화면에서 download 클릭
   (2) Eclipse/Aptana 메뉴 클릭
   (3) Installation 항목대로 진행
   (4) 플러그인 설치 완료 후 이클립스 다시 시작
